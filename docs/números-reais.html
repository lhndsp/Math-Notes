1.1 Axiomas e teoremas

(Anton and Rorres 2000)

1.1.1 Corpo

Defini-se como corpo qualquer conjunto \(\mathbb{K}\) no qual estão definidas as operações de soma e multiplicação que satisfazem as propriedades:

\[\begin{equation} \forall x \in \mathbb{R} ~~ \exists! ~~ a,b,c,d \in \mathbb{R} \mid x = a + b, x = cd\tag{1.1} \end{equation}\] \[\begin{equation} (a + b) + c = a + (c + b) \text{ e } (ab)c = a(bc)\tag{1.2} \end{equation}\] \[\begin{equation} a + b = b + a \text{ e } ab = ba\tag{1.3} \end{equation}\] \[\begin{equation} a + 0 = a \text{ e } 1a = a\tag{1.4} \end{equation}\] \[\begin{equation} a (b + c) = ab + cd\tag{1.5} \end{equation}\] \[\begin{equation} \forall a \in \mathbb{R} \exists! b \in \mathbb{R} \mid a + b = 0\tag{1.6} \end{equation}\] \[\begin{equation} \forall a \in \mathbb{R} \exists! b \in \mathbb{R} \mid ab = 1\tag{1.7} \end{equation}\] \[\begin{equation} -(-a) = a\tag{1.8} \end{equation}\]

São exemplos de corpos os conjuntos \(\mathbb{Q}\), \(\mathbb{R}\) e \(\mathbb{C}\), sendo que \(\mathbb{N}\) não considerado um corpo por não ter definido o elemento neutro multiplicativo e aditivivo e \(\mathbb{Z}\) não é um corpo por não possuir recíproco.

Das propriedades definidas acima, pode-se deduzir os teoremas:

Teorema 1.1 (Simplificação para adição) \[ \forall a,b,c \in \mathbb{R} \mid a+b=a+c \Rightarrow b = c \]

Prova. Por (1.6) temos que \(\exists! y \in \mathbb{R} \mid a+y = 0 \Rightarrow y+a+b=y+a+c\) e por (1.3) \(0+b = 0+c \Rightarrow b = c\)

Teorema 1.2 (Subtração) \[ \forall a,b \in \mathbb{R} \exists! x \in \mathbb{R} \mid a + (-x) = b, \quad a-x=b \]

Prova. Sabe-se que \(\exists!y \mid a+y = 0\), esse \(y\) é chamado \(-a\) logo \(a+(-x)+(-a)=b+(-a) \Leftrightarrow -x = b + (-a)\), então podemos reescrever a equação como \(a + b + (-a) = a + b + (-a) \equiv b = b\) que é verdadeira.

Teorema 1.3 (Divisão) \[ \forall a,b \in \mathbb{R} \exists! x \in \mathbb{R}^* \mid xb = a, \quad \frac{a}{x}=b, \quad \frac{a}{b} = x \]

Prova. \(\exists!y\mid by=1 \Rightarrow xby=ay \Rightarrow x = ay \Rightarrow ayb = ayb\) Sendo que essa ultima igualdade resulta em uma expressão verdadeira

Teorema 1.4 (Frações iguais) \[ a,c \in \mathbb{R} \wedge b,d \in \mathbb{R}^* \Rightarrow \frac{a}{b} = \frac{c}{d} \Leftrightarrow ad = bc \]

Prova. Temos que \(\frac{a}{b}=\frac{c}{d} \Leftrightarrow \frac{a}{b} - \frac{c}{d} = 0\) e por 1.6 \(\frac{a}{b} - \frac{c}{d} = \frac{ab - bc}{bd} = 0 \Leftrightarrow ab - bc = 0 \equiv ad = bc\)

Teorema 1.5 (Recíprocos de frações) \[ a, b \in \mathbb{R}^* \Rightarrow \frac{1}{\frac{a}{b}} = \frac{b}{a} \]

Prova (Demonstração). Por (1.7) \(\exists!x \in \mathbb{R}^* \mid \frac{a}{b}x = 1 \Leftrightarrow ax = b\) Logo \(x = \frac{b}{a}\)

Teorema 1.6 (Soma e subtração de frações) \[ a,c \in \mathbb{R} \wedge b,d \in \mathbb{R}^* \Rightarrow \frac{a}{b} + \frac{c}{d} = \frac{ad + bc}{bd} \quad \text{e} \quad \frac{a}{b} - \frac{c}{d} = \frac{ad - bc}{bd} \]

1.1.2 Ordem

Um corpo \(K\) é dito ordenado se apartir de seu eixo de simetria, chamado \(0\), e seus subconjuntos \(K^+|\forall k \in K \Rightarrow a > 0\) e \(K^-|\forall k \in K \Rightarrow a < 0\), quaisquer dos elementos \(a, b, c \in K\), satisfazem as propriedades:

\[\begin{equation} a,b \in \mathbb{R} \Leftrightarrow \left\{\begin{matrix} a < b\\ a > b \\ a = b \end{matrix}\right. \tag{1.9} \end{equation}\]

Teorema 1.8 \[ a, b \in \mathbb{R}^+ \Rightarrow a + b \in \mathbb{R}^+ \]

Teorema 1.9 \[ a, b \in \mathbb{R}^* \mid a < b \Rightarrow a-b<0 \]

Teorema 1.10 \[ \left\{\begin{matrix} a,b \in \mathbb{R}^+ \Rightarrow ab \in \mathbb{R}^+ \\ a \in \mathbb{R}^+, b \in \mathbb{R}^- \Rightarrow ab \in \mathbb{R}^- \\ a \in \mathbb{R}^-, b \in \mathbb{R}^+ \Rightarrow ab \in \mathbb{R}^- \\ \end{matrix}\right. \]

Definição 1.1 \[ a,b \in \mathbb{R}^- \Rightarrow ab \in \mathbb{R}^+ \]

Teorema 1.11 \[ \forall a,b \in \mathbb{R}^* \left\{\begin{matrix} |a| < |b| \Rightarrow \frac{a}{b} \in \left]-1,1 \right[\ \\ |a| > |b| \Rightarrow \frac{a}{b} \notin \left]-1,1 \right[\ \\ |a| = |b| \Rightarrow \frac{a}{b} = 1 \end{matrix}\right. \]

Teorema 1.12 \[ a, b, c \in \mathbb{R} \mid a < b \Rightarrow a+c < b+c \]

Teorema 1.13 \[ a, b \in \mathbb{R}, c \in \mathbb{R}^- \mid a < b \Rightarrow ac > bc \]

Teorema 1.14 \[ a, b \in \mathbb{R}^* \mid a < b \Rightarrow -a > -b \]
